TraceInv.ComputeTraceOfInverse¶

TraceInv.ComputeTraceOfInverse(A, ComputeMethod='cholesky', **Options)[source]¶

Computes the trace of inverse of a matrix without using interpolation.

The trace of inverse is computed with one of these three methods in this function.

ComputeMethod

Type

Cholesky

exact

Hutchinson

approximation

Stochastic Lanczos Quadrature

approximation
Parameters

  • A (ndarray) – Invertible matrix

  • ComputeMethod (string) – One of 'cholesky', 'hutchinson', or 'SLQ'. Default if 'cholesky'.

  • Options ('**kwargs') – Options for either of the methods.

Returns

trace of inverse of matrix

Return type

float

Raises

RunTimeError – Method is not recognized.
