Computes the trace of inverse of an invertible matrix \(\mathbf{A} + t \mathbf{B}\) using eigenvalues of \(\mathbf{A}\) and \(\mathbf{B}\) by

\[\mathrm{trace}\left( (\mathbf{A} + t \mathbf{B})^{-1} \right) = \sum_{i = 1}^n \frac{1}{\lambda_i + t \mu_i}\]

where \(\lambda_i\) is the eigenvalue of \(\mathbf{A}\) and \(\mu_i\) is the eigenvalue of \(\mathbf{B}\). This class does not accept interpolant points as the result is not interpolated.

Parameters

  • A (numpy.ndarray) – Invertible matrix, can be rither dense or sparse matrix.

  • B (numpy.ndarray) – Invertible matrix, can be either dense or sparse matrix.

  • ComputeOptions (dict) – A dictionary of input arguments for TraceInv.ComputeTraceOfInverse.ComputeTraceOfInverse module.

  • NonZeroRatio (int) – The ratio of the number of eigenvalues to be assumed non-zero. Used for sparse matrices. Default is 0.9 indicating to compute 90 percent of the eigenevalues with the largest magnitude and assume the rest of the eigenvalues are zero.

  • Tol (float) – Tolerance of computing eigenvalues. Used onlt for sparse matrices. Default value is 1e-3.

Note

The input matrices \(\mathbf{A}\) and \(\mathbf{B}\) can be either sparse or dense. In case of a sparse matrix, only some of the eigenvalues with the largest magnitude is computed and the rest of its eigenvalues is assumed to be negligible. The ratio of computed eigenvalues over the total number of eigenvalues can be set by NonZeroRatio. The tolerance at which the eigenvalues are computed can be set by Tol.

Example

This class can be invoked from the TraceInv.InterpolateTraceOfInverse.InterpolateTraceOfInverse module using InterpolationMethod='EIG' argument.

from TraceInv import GenerateMatrix
from TraceInv import InterpolateTraceOfInverse

# Generate a symmetric positive-definite matrix of the shape (20**2,20**2)
A = GenerateMatrix(NumPoints=20)

# Create an object that interpolats trace of inverse of A+tI (I is identity matrix)
TI = InterpolateTraceOfInverse(A,InterpolatingMethod='EIG')

# Interpolate A+tI at some input point t
t = 4e-1
trace = TI.Interpolate(t)

See also

The result of the EIG method is identical with the exact method EXT, which is given by TraceInv.InterpolateTraceOfInverse.ExactMethod.

This function sets the following class attirbutes:

LowerBound(t)[source]¶

Computes the lower bound of the trce of the inverse of the one-parameter function A+tB.

The lower bound is given by Remark 2.2 of [Ameli-2020] as

\[\mathrm{trace}((\mathbf{A}+t\mathbf{B})^{-1}) \geq \frac{n^2}{\mathrm{trace}(\mathbf{A}) + \mathrm{trace}(t \mathbf{B})}\]
ScaleInterpolantPoints()[source]¶

If the largest interpolant point in self.InterpolantPoints is greater than 1, this function rescales their range to max at 1. This function is intended to be used internally.

UpperBound(t)[source]¶

Computes the upper bound of the trace of the inverse of the one-parameter function A+tB.

The upper bound is given by Theorem 1 of [Ameli-2020] as

\[\frac{1}{\tau(t)} \geq \frac{1}{\tau_0} + t\]

where

\[\tau(t) = \frac{\mathrm{trace}\left( (\mathbf{A} + t \mathbf{B})^{-1} \right)}{\mathrm{trace}(\mathbf{B}^{-1})}\]

and \(\tau_0 = \tau(0)\).

Computes the trace of inverse of an invertible matrix \(\mathbf{A} + t \mathbf{B}\) using an interpolation scheme based on root monomial basis functions (see details below).

Parameters

  • A (numpy.ndarray) – Invertible matrix, can be rither dense or sparse matrix.

  • B (numpy.ndarray) – Invertible matrix, can be either dense or sparse matrix.

  • ComputeOptions (dict) – A dictionary of input arguments for TraceInv.ComputeTraceOfInverse.ComputeTraceOfInverse module.

  • NonZeroRatio (int) – The ratio of the number of eigenvalues to be assumed non-zero. Used for sparse matrices. Default is 0.9 indicating to compute 90 percent of the eigenevalues with the largest magnitude and assume the rest of the eigenvalues are zero.

  • Tol (float) – Tolerance of computing eigenvalues. Used onlt for sparse matrices. Default value is 1e-3.

Interpolation Method

Define the function

\[\tau(t) = \frac{\mathrm{trace}\left( (\mathbf{A} + t \mathbf{B})^{-1} \right)}{\mathrm{trace}(\mathbf{B}^{-1})}\]

and \(\tau_0 = \tau(0)\). Then, we approximate \(\tau^{-1}(t)\) by

\[\frac{1}{\tau(t)} = \frac{1}{\tau_0} + \sum_{i = 0}^p w_i \phi_i(t),\]

where \(\phi_i\) are basis functions defined by

\[\phi_i(t) = t^{\frac{1}{i+1}}, \qquad i = 0,\dots,p\]

and \(w_i\) are the coefficients of the linear basis functions with \(w_{0} = 1\) and the rest of the weights are to be found form the known function values \(\tau_i = \tau(t_i)\) at some given interpolant points \(t_i\).

Example

This class can be invoked from the TraceInv.InterpolateTraceOfInverse.InterpolateTraceOfInverse module using InterpolationMethod='EIG' argument.

from TraceInv import GenerateMatrix
from TraceInv import InterpolateTraceOfInverse

# Generate a symmetric positive-definite matrix of the shape (20**2,20**2)
A = GenerateMatrix(NumPoints=20)

# Create an object that interpolats trace of inverse of A+tI (I is identity matrix)
TI = InterpolateTraceOfInverse(A,InterpolatingMethod='EIG')

# Interpolate A+tI at some input point t
t = 4e-1
trace = TI.Interpolate(t)

See also

The result of the EIG method is identical with the exact method EXT, which is given by TraceInv.InterpolateTraceOfInverse.ExactMethod.
