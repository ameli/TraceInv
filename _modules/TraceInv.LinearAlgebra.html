TraceInv.LinearAlgebra.MatrixReduction module¶

TraceInv.LinearAlgebra.MatrixReduction.GolubKahnLanczosBidiagonalization(A, w, m, Tolerance=1e-08)[source]¶

B-diagonalizes the positive-definite matrix A using Golub-Kahn-Lanczos method.

This method bi-diagonalizes matrix A to B using the start vector w. m is the Lanczos degree, which will be the size of square matrix B.

Note

  • A should be positive-definite matrix size n*n.

  • B will be positive-definite matrix of size (m+1,m+1).

Warning

When matrix A is very close to the identity matrix, the Golub-Kahn bi-doagonalization method can not find \(\beta\), as \(\beta\) becomes zero. If A is not exactly identity, you may decrease the Tolerance to a very small number. However, if A is almost identity matrix, decreasing tolerance will not help, and this function cannot be used.

Parameters

  • A (numpy.ndarray) – Input matrix of the size n*n. Matrix should be positive-definite and symmetric.

  • w (array) – Start vector for the Lanczos tri-diagonalization. Column vector of size n. It could be generated randomly. Often it is generated by the Rademacher distribution with entries +1 and -1.

  • m (int) – Lanczos degree, which is the number of Lanczos iterations.

  • Tolerance (float) – The tolerance of the residual error of the Lanczos iteration.

Returns

Symmetric positive-definite matrix B of the size (m+1)*(m+1).

Return type

numpy.ndarray

Reference:

  • NetLib Algorithm 6.27

  • Matrix Computations, Golub, p. 495

  • Templates for Solution of Algebraic Eigenvalue Problem, Demmel, p. 143

TraceInv.LinearAlgebra.SparseCholesky module¶

TraceInv.LinearAlgebra.SparseCholesky.SparseCholesky(A, DiagonalOnly=False)[source]¶

This function uses LU decomposition assuming that A is symmetric and positive-definite.

Note

This function does not check if A is positive-definite. If the input matrix is not positive-definite, the Cholesky decomposition does not exist and the return value is misleadingly wrong.

Parameters

  • A (numpy.ndarray) – Symmetric and positive-definite matrix.

  • DiagonalOnly – If True, returns a column array of the diagonals of the Cholesky decomposition. If False, returns the full Cholesky matrix as scipy.sparse.csc_matrix.

Returns

Cholesky decomposition of A.

Return type

Super.LU
