Specifies general settings for the plots in the example scripts, namely, it sets plot themes by seaborn, fonts by LaTeX if available.

TraceInv.Utilities.PlotUtilities.SavePlot(plt, Filename, TransparentBackground=True)[source]¶

Saves plots.

Parameters

  • plt (matplotlib.pyplot) – matplotlib.pyplot object for the plots.

  • Filename (string) – Name of the file without extension or directory name.

Format:

The file is saved in both svg and pdf format.

Directory:

The plot is saved in the directory /docs/images/ with respect to the package root, if this directory is exists and writable. Otherwise, the plot is saved in the current directory of the user.
