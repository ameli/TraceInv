ComputeLogDeterminant¶

TraceInv.ComputeLogDeterminant.ComputeLogDeterminant(A, ComputeMethod='cholesky', NumIterations=20, LanczosDegree=20, UseLanczosTridiagonalization=False)[source]¶

Computes the log-determinan of full-rank matrix A.

Parameters

  • A (numpy.ndarray or scipy.sparse.csc_matrix) – A full-rank matrix.

  • NumIterations (int) – Number of Monte-Carlo trials

  • LanczosDegree (int) – Lanczos degree of the tri-diagonalization (or bi-diaqgonalization) process

  • UseLanczosTridiagonalization (bool) – Flag, if True, it uses the Lanczos tridiagonalization. If False, it uses the Golub-Kahn bi-diagonalization.

Returns

Log-determinant of A

Return type

float

Note

For computing the trace of inverse with the sothcastic Lanczos quadrature method (see TraceInv.ComputeTraceOfInverse.StochasticLanczosQuadrature), the preferred algorithm is the Lanczos tri-diagonalization, as opposed to Golub-Kahn bi-diagonalization.

In contrast to the above, the preferred SLQ method for computing log-determinant is the Golu-Kahn bi-diagonalization. The reason is that if the matrix \(\mathbf{A}\) has many singular values close to zero, bi-diagonalization performs better, and this matters when we compute determinant.

References:

  • Ubaru, S., Chen, J., and Saad, Y. (2017). Fast estimation of \(\mathrm{tr}(f(A))\) via stochastic Lanczos quadrature. SIAM Journal on Matrix Analysis and Applications, 38(4), 1075-1099. doi: 10.1137/16M1104974

Examples:

>>> # Import packages
>>> from TraceInv import ComputeLogDeterminant
>>> from TraceInv import GenerateMatrix

>>> # Generate a sample matrix
>>> A = GenerateMatrix(NumPoints=20)

>>> # Compute log-determinant with Cholesky method
>>> LogDet_1 = ComputeLogDeterminant(A)

>>> # Compute log-determinant with stochastic Lanczos quadrature method
>>> LogDet_1 = ComputeLogDeterminant(A,ComputeMethod='SLQ',NumIterations=20,LanczosDegree=20)

>>> # Compute log-determinant with stochastic Lanczos quadrature method with Golub-Khan bi-diagonaliation
>>> LogDet_1 = ComputeLogDeterminant(A,ComputeMethod='SLQ',NumIterations=20, 
...                 LanczosDegree=20,Tridiagonalization=False)
