ComputeTraceInvBySolvingLinearSystem¶

TraceInv.ComputeTraceOfInverse.CholeskyMethod.ComputeTraceInvBySolvingLinearSystem(L, n, UseSparse)[source]¶

Computes the trace of inverse by solving a linear system for Cholesky matrix and each column of the identity matrix to obtain the inverse of L sub-sequentially.

The matrix \(\mathbf{L}\) is not inverted directly, rather, the linear system

\[\mathbf{L} \boldsymbol{x}_i = \boldsymbol{e}_i, \qquad i = 1,\dots,n\]

is solved, where \(\boldsymbol{e}_i = (0,\dots,0,1,0,\dots,0)^{\intercal}\) is a column vector of zeros except its \(i\) th entry is one, and \(n\) is the size of the square matrix \(\mathbf{A}\). The solution \(\boldsymbol{x}_i\) is the \(i\) th column of \(\mathbf{L}^{-1}\). Then, its Frobenius norm is

\[\| \mathbf{L} \|_F^2 = \sum_{i=1}^n \| \boldsymbol{x}_i \|^2.\]

The method is memory efficient as the vectors \(\boldsymbol{x}_i\) do not need to be stored, rather, their norm can be stored in each iteration.

Note

This method is slow, and it should be used only if the direct matrix inversion can not be computed (such as for large matrices).

Parameters

  • L (ndarray) – Cholesky matrix

  • UseSparse (bool) – Flag, if true, the matrix L is considered as sparse.

Returns

Trace of matrix A.

Return type

float
