LinearSolver¶

TraceInv._LinearAlgebra.LinearSolver.LinearSolver(A, b, Tol=1e-05)[source]¶

Solves the linear system \(Ax = b\) where \(A\) can be either sparse or dense.

Parameters

  • A (numpy.ndarray) – matrix of coefficients, two-dimensional array, can be either sparse or dense

  • b (array) – column vector of the right hand side of the linear system, one-dimensional array

  • Tol (float) – Tolerance for the error of solving linear system. This is only applicable if A is sparse.

Returns

one-dimensional array of the solution of the linear system

Return type

numpy.array
